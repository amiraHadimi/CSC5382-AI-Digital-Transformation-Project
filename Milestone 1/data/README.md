# Dataset Description

This folder contains a small sample extracted from the Agile User Story Point Estimation dataset
introduced by Choetkiertikul et al. (2016).

The full dataset contains 23,313 issues from 16 open-source projects and is publicly available at:
https://github.com/mrthlinh/Agile-User-Story-Point-Estimation

The sample is provided for illustration and reproducibility purposes.
